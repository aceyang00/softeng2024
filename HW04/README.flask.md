# 과제 #04 웹 어플리케이션 만들기

#### Flask를 사용해 웹으로 전송

#### Get방식 
- GET 방식은 URL에 데이터가 그대로 노출되기 때문에 개인정보를 전송할때는 사용하지 않는것이 좋음.
- form method="GET" action="/calculate"

 = 데이터가 입력되면 get방식으로 /calculate 경로에 데이터 전송
